MacOS自带的`Activity Monitor`，如果你想在menu bar上可以时时
监控，推荐下面这个开源app。相比较其他app，我最喜欢这款app可以显示数字进度，不是用bar。

### Stats
* App：https://github.com/exelban/stats
* 效果图
  ![Alt text](/images/posts/tools/stats.png)

### 特点
* CPU 使用
* GPU 使用
* 内存使用
* 硬盘使用
* 传感器(温度/电压/电力)
* 电池使用
* 网络速度
* 风扇速度
